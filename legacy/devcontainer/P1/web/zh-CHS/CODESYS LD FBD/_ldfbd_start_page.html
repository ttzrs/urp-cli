CODESYS LD FBD

当CODESYS LD FBD在您的开发环境中安装插件后，您可以根据 IEC 61131-3，使用 LD 或 FBD 语言以图形方式为控制器应用程序创建编程对象。

LD代表梯形逻辑图 .编程语言以图形方式表示继电器的经典接线。

功能块图代表功能框图 . FBD 中的编程块也称为功能块语言 (FBD)。图形化编程语言允许使用由系统、自身编程或库提供的运算块进行编程。

LD 或 FBD 中实现的编程对象可以切换到其他编程语言。此外，还可以切换到现在的已弃用 IL编程语言。

IL代表指令列表并用于控制器输入和输出的逻辑链接。

您可以使用CODESYS Installer更新CODESYS LD FBD添加在。

提示

IL可以在CODESYS如果需要的话，选项：在FBD、LD 和IL对话，在IL选项卡，选择启用IL选项。
