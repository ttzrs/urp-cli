适用于 Android 的 CODESYS 传感器应用程序

用 CODESYS Sensor App​ for Android 产品，来自的传感器数据可以将安卓智能手机发送到 CODESYS 控制器。

产品描述

智能手机集成了许多传感器，例如霍尔效应、加速度计和GPS传感器。 CODESYS Sensor App​ for Android，来自最常见传感器的数据可以从基于 Android 的设备发送到 CODESYS 通过网络连接控制器。此外，您可以使用Android的语音识别将识别出的单词作为字符串发送到控制器。这样，当互联网连接可用时，你可以在你的安卓设备（例如智能手机）上实现简单的语音控制。

该软件包包含的示例项目 CODESYS Development System 这允许在可视化中显示发送的数据。

其他信息

. 软件包的内容：

  • CODESYS 用于对数据进行可视化的示例项目 CODESYS 控制器

  • Android Sensor Netvar 包含一些辅助数据结构的库

. 传输以下传感器数据：

  • 加速

  • 陀螺仪

  • 磁场

  • 旋转

  • 接近

  • 温度

  • 湿度

  • 光强度

  • 气压

  • 通过语音识别识别的单词

可用的传感器数据可能因智能手机型号而异。

有关更多信息，另请参阅 适用于 Android 的传感器应用程序 示例项目。
