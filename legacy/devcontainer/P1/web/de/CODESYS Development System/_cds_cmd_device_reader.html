Befehl: Device License Reader

Funktion: Der Befehl öffnet den Standarddialog Gerät auswählen und liest nach Auswahl einer Steuerung die Lizenz- und Produktinformationen dieser Steuerung aus. Diese Lizenz- und Produktinformationen werden im Dialog Device License Reader angezeigt.

Aufruf: Menü Tools

Voraussetzung: Auf der Steuerung befindet sich keine Applikation.

Tipp

Wenn der Befehl gewählt wird, obwohl sich auf der Steuerung eine Applikation befindet, wird der Anwender in einem Dialog gefragt, ob alle Applikationen von der Steuerung entfernt werden sollen. Wenn der Benutzer in diesem Dialog auf Nein klickt, wird der Befehl Device License Reader abgebrochen.
