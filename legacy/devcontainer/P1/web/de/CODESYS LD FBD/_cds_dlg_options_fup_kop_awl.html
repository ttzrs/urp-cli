Dialog: Optionen: FUP, KOP und AWL

Symbol:

Funktion: Der Dialog dient der Konfiguration der Darstellungsoptionen für den FUP/KOP/AWL-Editor.

Aufruf: Befehl Tools → Optionen, Kategorie FUP, KOP und AWL

Registerkarte: Allgemeines

Tabelle 7. Ansicht

Netzwerktitel anzeigen

Der Netzwerktitel wird in der oberen linken Ecke des Netzwerks angezeigt.

Netzwerkkommentar anzeigen

Das Netzwerkkommentar wird in der oberen linken Ecke des Netzwerks angezegt. Wenn CODESYS zusätzlich den Netzwerktitel darstellt, erscheint der Kommentar in der Zeile unterhalb.

Bausteinsymbol anzeigen

Das Bausteinsymbol wird im Bausteinelement im FUP- und KOP-Editor angezeigt. Auch die Standardoperatoren haben Symbole.

Operandenkommentar anzeigen

CODESYS zeigt den Kommentar an, den Sie einer Variablen im Implementierungsteil gegeben haben. Der Operandenkommentar bezieht sich nur auf die lokale Verwendungsstelle der Variablen, im Gegensatz zum „Symbolkommentar“.

Der Kommentar wird abhängig vom verfügbaren Platz automatisch umgebrochen.

Sie können den Kommentar auf eine definierte Breite begrenzen indem Sie die Option Feste Größe für Operandenfelder aktivieren.

Symbolkommentar anzeigen

CODESYS zeigt den Kommentar, den Sie einer Variablen oder einem Symbol bei der Deklaration gegeben haben, oberhalb des Variablennamens an. Zusätzlich oder anstelle des Symbolkommentars können Sie auch einen lokalen „Operandenkommentar“ zuweisen.

Symboladresse anzeigen

Wenn einem Symbol (Variable) eine Adresse zugewiesen ist, wird diese Adresse oberhalb des Variablennamens angezeigt.

Netzwerktrennlinien anzeigen

Zwischen den einzelnen Netzwerken wird eine Trennlinie angezeigt.
