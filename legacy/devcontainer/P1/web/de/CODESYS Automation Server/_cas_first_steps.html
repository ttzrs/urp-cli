Erste Schritte mit dem CODESYS Automation Server

Für die Verwendung der CODESYS Automation Server-Plattform zur Kommunikation mit Steuerungen und mit dem CODESYS Development System benötigen Sie die folgenden Produkte, die im CODESYS Store International erhältlich sind:

  • CODESYS Automation Server

    Damit erhalten Sie Ihren Zugang zur Plattform CODESYS Automation Server, um vom CODESYS Automation Server aus mit Steuerungen des Netzwerks zu kommunizieren

  • CODESYS Automation Server Connector

    Add-on für das CODESYS Development System, um vom CODESYS Development System aus mit dem CODESYS Automation Server zu kommunizieren

  • CODESYS Edge Gateway

    Erweitertes CODESYS Gateway für die Kommunikation zwischen dem Steuerungsnetzwerk und dem CODESYS Automation Server

Wichtig

Beachten Sie unbedingt die Sicherheitshinweise zur Verwendung des CODESYS Edge-Gateway!

Tipp

Wenn Sie das Package CODESYS Edge Gateway auf Ihrem Computer installieren, werden die Services des bereits existierenden Gateways überschrieben.

Hinweis

CODESYS Automation Server im CODESYS Store International bestellen

  1. Öffnen Sie den CODESYS Store International oder den CODESYS Store North America.

  2. Im CODESYS Store International wählen Sie die Registerkarte CODESYS Automation Server.

  3. Klicken Sie auf das Produkt CODESYS Automation Server.

    Ausführliche Produktinformationen werden angezeigt.

  4. Geben Sie in das Feld Hostname (wird in der URL genutzt): einen eindeutigen Namen für Ihren CODESYS Automation Server ein.

    Für Plattform und Region werden in einer späteren Version Auswahllisten zur Verfügung stehen.

  5. Klicken Sie auf Jetzt bestellen.

  6. Im nächsten Schritt geben Sie die Zahlungsdaten ein und akzeptieren die AGB und Dateschutzbestimmungen der Firma CODESYS GmbH sowie die Einverständniserklärung, dass Ihre Geschäftsdaten für bestimmte Zwecke an Drittanbieter weitergegeben werden dürfen.

  7. Um den Bestellvorgang abzuschließen, klicken Sie nun auf Jetzt kaufen.

  8. Anschließend erhalten Sie eine E-Mail mit der Bestätigung Ihrer Bestellung und eine E-Mail mit dem Aktivierungslink für Ihren CODESYS Automation Server „<host name>“.

    In der Bestätigungsmail finden Sie einen Link zu Ihrem Benutzerkonto, auf dem Sie stets die aktuellen Artikeldaten und Kosteninformationen zu Ihrem CODESYS Automation Server einsehen können. In dem Benutzerkonto können Sie auch Änderungen und eine Kündigung Ihres CODESYS Automation Server vornehmen.

  9. In der E-Mail mit dem Aktivierungslink klicken Sie auf den Link Setup user.

    Der Dialog Add initial admin account des CODESYS Automation Server öffnet sich. Im Dialog wird auch die ID Ihres Tickets angezeigt.

  10. Geben Sie im Dialog Add initial admin account den Name, im Feld User name Ihren Benutzernamen und Ihre E-Mail-Adresse ein und vergeben Sie ein sicheres eindeutiges Passwort. Das Passwort muss folgende Anforderungen erfüllen:

    • Mindestlänge 8 Zeichen

    • Mindestens 6 unterschiedliche Zeichen

    • Mindestens eine Ziffer (0..9)

    • Mindestens ein Großbuchstabe

    • Mindestens ein Kleinbuchstabe

    Anschließend klicken Sie auf die Schaltfläche Create initial administrator.

    Im nächsten Fenster werden die Meldung Initial administrator account was created und die URL für das Einloggen auf Ihren Server angezeigt.

    Tipp

    Kopieren Sie sich diese URL beispielsweise auf den Desktop Ihres Computers, da Sie über diese URL auf die Login-Seite Ihres Automation Servers gelangen.

    Ihre E-Mail-Adresse und das vergebene Passwort benötigen Sie auch, um sich vom CODESYS Development System mit Ihrem Automation Server zu verbinden.

  11. Kopieren Sie diese URL und fügen Sie diese in Ihrem Browser ein.

    Jetzt können Sie sich auf der Seite Anmelden mit Ihrem Benutzernamen und dem im vorigen Schritt vergebenen Passwort auf Ihrem Server einloggen.

  12. Anschließend können Sie gemäß den Anleitungen dieser Hilfe auf dem CODESYS Automation Server Steuerungen eintragen und den Server mit diesen verbinden. Vom CODESYS Development System können Sie nun mit Hilfe des Packages CODESYS Automation Server Connector dem Server ein Edge-Gateway hinzufügen und CODESYS-Projekte und Bootapplikationen auf den CODESYS Automation Server laden.
