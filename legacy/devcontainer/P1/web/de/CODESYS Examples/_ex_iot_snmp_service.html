Beispiel: Verwenden der Bibliothek SNMP Service SL

Produkt: CODESYS IIOT Libraries SL

Das Beispielprojekt SNMP Service SL Example.project, das auch eine Visualisierung für das Senden und Empfangen von SNMP-Nachrichten bereitstellt, veranschaulicht die Verwendung der Bibliothek SNMP Service SL.

Beschreibung

Das Beispielprojekt SNMP Service SL Example.project veranschaulicht die Verwendung der Bibliotheksbausteinen in den folgenden Applikationen.

Applikation SNMPAgentExample

Das Beispiel zeigt, wie der Bibliotheksbaustein SNMP_AGENT verwendet wird und wie Werte registriert werden können. Eine Visualisierung zeigt den Status des Agenten an.

Applikation SNMPGetNextExample

Das Beispiel zeigt die Verwendung des Bibliotheksbausteins SNMP_GET_REQUEST für eine SNMP GET_NEXT-Anfrage. Die Ergebniswerte einer Anfrage werden in einer Visualisierung dargestellt.

Bitte beachten Sie, dass im Beispiel der zugehörige Wert einer OID immer im nachfolgenden Datensatz steht.

Die IP-Adressen und die OID werden über die Visualisierung konfiguriert.

Applikation SNMPSetExample

Das Beispiel veranschaulicht die Verwendung des Bibliotheksbausteins SNMP_SET. Über eine Visualisierung können SET-Telegramme versendet werden.

Applikation SNMPTrapReceiver

Das Beispiel veranschaulicht die Verwendung des Bibliotheksbaustein SNMP_TRAP_RECEIVER. Die empfangenen Werte werden in einer Visualisierung dargestellt.

Applikation SNMPTrapSender

Das Beispiel veranschaulicht die Verwendung des Bibliotheksbaustein SNMP_TRAP_SENDER. Über eine Visualisierung können TRAP-Telegramme verschickt werden.
