Beispiel: Umschalten zwischen mehreren Visualisierungen

Produkt: CODESYS Visualization

Das Beispiel zeigt, wie Sie zwischen Visualisierungen, die in einem Frame referenziert werden, umschalten können.

Beschreibung

In dem Beispielprojekt VisualizationSwitching werden fünf Implementierungsoptionen dargestellt.

Weitere Informationen

Das Projekt VisualizationSwitching umfasst fünf Frames mit den folgenden Funktionen:

  • Frame 1: Manuelle Umschaltung zwischen den Visualisierungen

  • Frame 2: Automatisches programmgesteuertes Umschalten der Visualisierungen via Variable.

  • Frame 3: Automatisches programmgesteuertes Umschalten der Visualisierungen über den FrameManager.

  • Frame 4: Manuelle Änderung der Visualisierung via Variable.

  • Frame 5: Automatisches, programmgesteuertes Umschalten zwischen den Visualisierungen über eine Variable für jeden zweiten Client

Systemvoraussetzungen und Einschränkungen

Programmiersystem

CODESYS Development System Version 3.5.14.0 oder höher

Laufzeitsystem

CODESYS Control Win Version 3.5.14.0
