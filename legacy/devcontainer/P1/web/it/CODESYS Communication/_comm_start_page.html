CODESYS Communication

CODESYS Communication consente la semplice pubblicazione e il collegamento dei dati nelle applicazioni.

I dati vengono pubblicati tramite OPC UA. La funzione Communication Manager offre due opzioni a questo scopo:

  • Usando Modelli informativi OPC UA

    Questi sono supportati genericamente. I contenuti del modello informativo sono tradotti nei corrispondenti tipi di dati e dichiarazioni IEC.

  • Usando il Configurazione del set di simboli IEC del Responsabile della Comunicazione

    I simboli IEC possono essere rilasciati rapidamente e facilmente senza dover creare prima esplicitamente un modello informativo.

CODESYS Scripting

CODESYS Scripting consente di automatizzare comandi o complesse operazioni di programma in CODESYS, che altrimenti richiedono clic manuali del mouse e immissione di testo nel file CODESYS interfaccia utente. A tale scopo, gli script basati su (Iron)Python possono essere avviati dal file CODESYS interfaccia utente (comando o barra degli strumenti configurata) o dalla riga di comando di Windows.
