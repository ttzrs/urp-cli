CODESYS Visualization

Il CODESYS Visualization il componente aggiuntivo consente di creare interfacce utente completamente integrate CODESYS Development System in base agli elementi di visualizzazione disponibili. Questi vengono visualizzati su un dispositivo di destinazione, su un PC o nel browser web. Ciò è possibile anche contemporaneamente sulla base dello stesso set di dati.

È possibile creare interfacce utente strutturate che consistono in visualizzazioni interconnesse in qualsiasi modo complesso.

È possibile monitorare un'interfaccia utente insieme alla propria applicazione o utilizzarla per il test e la messa in servizio.

Funzionalità aggiuntive:

  • Istanziazione di blocchi funzione

  • Richiamo di funzioni IEC

  • Accesso all'array

  • Registrazione dei dati in tempo reale

  • Estensione della collezione di elementi di visualizzazione

Puoi scaricare CODESYS Visualization dal Negozio CODESYS. Lì troverai anche la scheda tecnica del prodotto.
