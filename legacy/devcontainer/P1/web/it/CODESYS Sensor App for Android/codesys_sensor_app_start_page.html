App CODESYS Sensor per Android

Con il CODESYS Sensor App​ for Android prodotto, i dati del sensore possono essere inviati da uno smartphone Android a un CODESYS controller.

Descrizione del prodotto

Gli smartphone integrano numerosi sensori come effetti Hall, accelerometri e sensori GPS. Usando CODESYS Sensor App​ for Android, i dati provenienti dai sensori più comuni possono essere inviati da dispositivi basati su Android a un CODESYS controller tramite una connessione di rete. Inoltre, puoi utilizzare il riconoscimento vocale di Android per inviare le parole riconosciute come stringa al controller. In questo modo, puoi implementare un semplice controllo vocale con il tuo dispositivo Android (ad esempio uno smartphone) quando è disponibile una connessione Internet.

Il pacchetto software contiene un progetto di esempio per CODESYS Development System che consente di visualizzare i dati inviati in una visualizzazione.

Informazioni aggiuntive

. Contenuto del pacchetto software:

  • CODESYS progetto di esempio per la visualizzazione dei dati su un CODESYS controller

  • Android Sensor Netvar libreria che include alcune strutture dati ausiliarie

. Vengono trasmessi i seguenti dati del sensore:

  • Accelerazione

  • giroscopio

  • Campo magnetico

  • Rotazione

  • Prossimità

  • Temperatura

  • Umidità

  • Intensità della luce

  • Pressione dell'aria

  • Parole riconosciute tramite riconoscimento vocale

I dati del sensore disponibili possono variare a seconda del modello di smartphone.

Per ulteriori informazioni, consulta anche la App Sensor per Android progetto di esempio.
