アンドロイド用CODESYSセンサーアプリ

と CODESYS Sensor App​ for Android 本製品からのセンサーデータをAndroidスマートフォンに送信できます CODESYS コントローラー。

商品説明

スマートフォンには、ホール効果、加速度計、GPSセンサーなどの多数のセンサーが組み込まれています。を使用する CODESYS Sensor App​ for Android、最も一般的なセンサーからのデータをAndroidベースのデバイスからに送信できます CODESYS ネットワーク接続経由のコントローラー。さらに、Android の音声認識を使用して、認識した単語を文字列としてコントローラーに送信できます。これにより、インターネットに接続できるときに Android デバイス（スマートフォンなど）で簡単な音声制御を実装できます。

ソフトウェアパッケージには、のサンプルプロジェクトが含まれています CODESYS Development System これにより、送信されたデータをビジュアライゼーションに表示できます。

追加情報

. ソフトウェアパッケージの内容:

  • CODESYS 上のデータを視覚化するためのサンプルプロジェクト CODESYS コントローラー

  • Android Sensor Netvar いくつかの補助データ構造を含むライブラリ。

. 次のセンサーデータが送信されます。

  • アクセラレーション

  • ジャイロスコープ

  • 磁場

  • ローテーション

  • 近接性

  • 温度

  • 湿度

  • 光強度

  • 空気圧

  • 音声認識で認識された単語

利用可能なセンサーデータは、スマートフォンのモデルによって異なる場合があります。
